# Timetable Clash Detection App

## Overview

This is a **React-based timetable analysis tool** that allows users to upload a timetable file and automatically generate a **clash report**. The application analyzes the schedule and identifies conflicts, helping users quickly see how many timetable clashes exist.

The goal of this project is to simplify timetable validation and make it easier to detect overlapping classes or scheduling issues.

---

## Features

* Upload timetable files
* Automatically analyze timetable data
* Detect and report scheduling clashes
* Display number of clashes in the timetable
* Simple and user-friendly interface

---

## Tech Stack

* **React.js**
* **JavaScript**
* **XLSX / Excel parsing**
* **HTML / CSS**

---

## How It Works

1. Upload a timetable file.
2. The application parses the timetable data.
3. The system analyzes time slots and detects overlapping entries.
4. A **clash report** is generated showing the number of conflicts.
